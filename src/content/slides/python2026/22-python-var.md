### 동적 타이핑

```python
x = 10          # int
x = "hello"     # str
x = [1, 2, 3]   # list
```

**장점**: 빠른 프로토타이핑  
**단점**: 런타임 에러 가능

### 다양한 데이터 타입

```python
name = "개발자"          # 문자열 (str)
age = 25                # 정수 (int)
height = 175.5          # 실수 (float)
is_student = True       # 불린 (bool)
fruits = ["사과", "바나나"]  # 리스트 (list)
person = {"name": "김파이", "age": 30}  # 딕셔너리 (dict)
```
