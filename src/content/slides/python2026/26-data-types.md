### 기본 데이터 타입

```python
# 숫자
integer = 42
floating = 3.14

# 문자열
text = "Hello"
multiline = """
여러 줄
문자열
"""

# 불린
is_active = True
is_done = False
```

### 컬렉션

```python
# 리스트 (순서 O, 변경 가능)
fruits = ["사과", "바나나", "체리"]

# 튜플 (순서 O, 변경 불가)
coordinates = (10, 20)

# 딕셔너리 (키-값 쌍)
user = {"name": "김파이", "age": 30}
```
