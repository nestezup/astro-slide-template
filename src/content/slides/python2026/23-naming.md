### snake_case (Python)

```python
# Good
user_name = "Alice"
total_price = 1000
is_valid_email = True

# Bad
userName = "Alice"      # camelCase
UserName = "Alice"      # PascalCase
```

### 상수는 대문자

```python
MAX_CONNECTIONS = 100
API_KEY = "abc123"
PI = 3.14159
```

### 타입 확인

```python
age = 25
print(type(age))  # <class 'int'>
```
