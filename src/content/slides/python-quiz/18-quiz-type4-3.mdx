<QuizQuestion number={18} title="반복문의 활용">
  정수 N을 입력받아 N단 구구단을 출력하세요.

  ### 출력 예시 (N=3일 때)
  ```text
  3 x 1 = 3
  3 x 2 = 6
  ...
  3 x 9 = 27
  ```

  ### 초기 코드
  ```python
  n = int(input())

  for i in range(1, 10):
      result = ____
      print(f"{n} x {i} = {result}")
  ```
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  n = int(input())

  for i in range(1, 10):
      result = n * i
      print(f"{n} x {i} = {result}")
  ```

  ### 해설
  1. `range(1, 10)`: 1부터 9까지 반복 (1, 2, ..., 9)
  2. `n * i`: 입력받은 단(n)과 1~9(i)를 곱해서 결과 저장
</QuizAnswer>
