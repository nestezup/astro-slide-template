<QuizQuestion number={2} title="변수와 자료형 변환">
  다음과 같이 출력되도록 빈칸을 채워 코드를 완성하세요.

  ### 출력 예시
  ```text
  Hello Python
  100
  Python50
  ```

  ### 초기 코드
  ```python
  msg = ____
  num = ____
  text = ____

  print(msg)
  print(num * 2)
  print(text + str(num))
  ```
  
  **힌트**: `str()` 함수는 숫자를 문자열로 변환합니다.
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  msg = "Hello Python"
  num = 50
  text = "Python"

  print(msg)             # "Hello Python"
  print(num * 2)         # 50 * 2 = 100
  print(text + str(num)) # "Python" + "50" = "Python50"
  ```

  ### 해설
  1. `msg`: 출력할 문자열 그대로 "Hello Python"
  2. `num`: 2를 곱해서 100이 되어야 하므로 50
  3. `text`: 숫자 50을 문자열로 바꾼 뒤 붙여서 "Python50"이 되려면 "Python"이어야 함
</QuizAnswer>
