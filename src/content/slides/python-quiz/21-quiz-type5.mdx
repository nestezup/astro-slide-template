<QuizAnswer>
  ### 정답 코드
  ```python
  def solution(route):
      east = 0
      north = 0
      for i in route:
          if i == "N":
              north += 1
          elif i == "S":
              north -= 1
          elif i == "E":
              east += 1
          elif i == "W":
              east -= 1
      return [east, north]
  ```

  ### 해설
  1. `i`가 "S"면 남쪽이므로 `north` 감소
  2. `i`가 "E"면 동쪽이므로 `east` 증가
  3. `i`가 "W"면 서쪽이므로 `east` 감소
</QuizAnswer>
