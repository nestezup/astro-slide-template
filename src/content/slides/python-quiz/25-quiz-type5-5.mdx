</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  def is_prime(n):
      if n < 2:
          return False
      for i in range(2, n):
          if n % i == 0:
              return False
      return True
  ```

  ### 해설
  1. 2부터 n-1까지의 수(`i`)로 `n`을 나누어 봄
  2. 나머지가 0인 경우가 하나라도 있으면 약수가 존재하므로 소수가 아님(`False`)
  3. 반복문을 끝까지 통과하면 소수(`True`)
</QuizAnswer>
