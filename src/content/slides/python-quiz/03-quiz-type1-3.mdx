<QuizQuestion number={3} title="f-string 문자열 포맷팅">
  f-string을 사용하여 다음과 같이 출력되도록 코드를 완성하세요.

  ### 출력 예시
  ```text
  이름: 김파이
  나이: 25세
  내년 나이: 26세
  ```

  ### 초기 코드
  ```python
  name = ____
  age = ____

  print(f"이름: {name}")
  print(f"나이: {age}세")
  print(f"내년 나이: {     }세")
  ```

  **힌트**: f-string 안에서 `{age + 1}` 처럼 연산식을 사용할 수 있습니다.
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  name = "김파이"
  age = 25

  print(f"이름: {name}")           # 이름: 김파이
  print(f"나이: {age}세")          # 나이: 25세
  print(f"내년 나이: {age + 1}세") # 내년 나이: 26세
  ```

  ### 해설
  1. `name`과 `age` 변수에 각각 값 할당
  2. f-string의 중괄호 `{}` 안에는 변수뿐만 아니라 연산식도 넣을 수 있음
  3. `age + 1`을 사용하여 내년 나이 계산
</QuizAnswer>
