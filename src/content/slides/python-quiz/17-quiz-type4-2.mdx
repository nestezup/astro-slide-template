<QuizQuestion number={17} title="for 반복문과 range">
  정수 N을 입력받아 1부터 N까지의 합을 계산하여 출력하세요.

  ### 초기 코드
  ```python
  n = int(input())

  total = 0
  for i in range(____, ____ + 1):
      total += ____

  print(total)
  ```

  **힌트**: `range(start, end)`는 `start`부터 `end-1`까지 생성합니다.
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  n = int(input())

  total = 0
  for i in range(1, n + 1):
      total += i

  print(total)
  ```

  ### 해설
  1. `range(1, n + 1)`: 1부터 n까지 숫자를 생성 (끝 값 포함 안 하므로 +1)
  2. `total += i`: 변수 i가 1, 2, 3... n으로 변하면서 total에 누적됨
</QuizAnswer>
