  정수 N의 팩토리얼(N!)을 계산하는 함수를 작성하세요.

  **정의**: $N! = N \times (N-1) \times ... \times 1$

  ### 초기 코드
  ```python
  def factorial(n):
      result = 1
      for i in range(1, n + 1):
          result *= ____
      return result

  # 테스트
  print(factorial(5))  # 120
  ```
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  def factorial(n):
      result = 1
      for i in range(1, n + 1):
          result *= i
      return result
  ```

  ### 해설
  1. `result`를 1로 초기화 (곱셈의 항등원)
  2. 1부터 n까지 반복하며 계속 곱함 (`*=`)
  3. `range(1, n + 1)` 사용
</QuizAnswer>
