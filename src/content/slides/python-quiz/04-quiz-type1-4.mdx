  ### 출력 예시
  ```text
  첫 번째 과일: 사과
  두 번째 과일: 바나나
  과일 개수: 3
  ```

  ### 초기 코드
  ```python
  fruits = ["사과", "바나나", "체리"]

  print(f"첫 번째 과일: {fruits[  ]}")
  print(f"두 번째 과일: {fruits[  ]}")
  print(f"과일 개수: {len(     )}")
  ```

  **힌트**: 리스트의 인덱스는 0부터 시작합니다.
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  fruits = ["사과", "바나나", "체리"]

  print(f"첫 번째 과일: {fruits[0]}")  # 0번 인덱스: 사과
  print(f"두 번째 과일: {fruits[1]}")  # 1번 인덱스: 바나나
  print(f"과일 개수: {len(fruits)}")   # len() 함수로 길이 구하기
  ```

  ### 해설
  1. 리스트의 첫 번째 요소 인덱스는 `0`
  2. 리스트의 두 번째 요소 인덱스는 `1`
  3. `len()` 함수에 리스트 이름을 넣어 전체 개수를 구함
</QuizAnswer>
