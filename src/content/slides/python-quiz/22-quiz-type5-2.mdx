  ```python
  def find_max(numbers):
      max_value = numbers[0]
      for num in numbers:
          # 현재 값(num)이 최댓값(max_value)보다 크다면
          if num > ____:
              max_value = ____
      return max_value

  # 테스트
  result = find_max([3, 7, 2, 9, 1])

  ```
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  def find_max(numbers):
      max_value = numbers[0]
      for num in numbers:
          if num > max_value:
              max_value = num
      return max_value
  ```

  ### 해설
  1. 첫 번째 값을 잠정적인 최댓값으로 설정
  2. 리스트를 순회하며 현재 `num`이 `max_value`보다 크면 교체
  3. 순회가 끝나면 `max_value`에 가장 큰 값이 남음
</QuizAnswer>
