<QuizQuestion number={10} title="실수형 연산">
  섭씨 온도가 주어질 때, 화씨 온도로 변환하여 출력하세요.

  **공식**: 화씨 = 섭씨 $\times 1.8 + 32$

  ### 초기 코드
  ```python
  celsius = float(input())

  fahrenheit = ____
  print(f"{celsius}°C = {fahrenheit:.1f}°F")
  ```
</QuizQuestion>

<QuizAnswer>
  ### 정답 코드
  ```python
  celsius = float(input())

  fahrenheit = celsius * 1.8 + 32
  print(f"{celsius}°C = {fahrenheit:.1f}°F")
  ```

  ### 해설
  1. 공식 그대로 코드로 옮김: `celsius * 1.8 + 32`
  2. 연산자 우선순위에 의해 곱셈(`*`)이 덧셈(`+`)보다 먼저 계산되므로 괄호 불필요
</QuizAnswer>
