# Bun이란?

Bun은 JavaScript 런타임입니다. Zig로 작성되었으며, Node.js와 호환되는 API를 제공합니다.

## 왜 Bun인가?

### **속도**
- 시작 시간이 3배 빠름
- 패키지 설치가 20배 빠름
- 실행 속도가 2-3배 향상

### **호환성**
- Node.js API 호환
- npm 패키지 사용 가능
- TypeScript 내장 지원

### **내장 도구**
- 패키지 매니저
- 번들러
- 테스트 러너

### **단일 바이너리**
- 하나의 파일로 설치
- 의존성 없음
- 크로스플랫폼 지원
