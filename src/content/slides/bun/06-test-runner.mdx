## Jest 호환, 하지만 더 빠름

`bun:test`는 Jest 호환 API를 제공합니다.  
테스트를 위해 무거운 의존성을 설치할 필요가 없습니다.

```typescript
import { describe, expect, test } from "bun:test";

describe("math", () => {
  test("adds 1 + 2 to equal 3", () => {
    expect(1 + 2).toBe(3);
  });
});
```
